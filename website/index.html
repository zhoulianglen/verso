Features

Open any .md file.
Start reading and writing.

Not a note manager. Not a knowledge base. Just the best way to open, read, and edit Markdown files on your machine.

WYSIWYG & Source Mode

Write in a rich visual editor or switch to raw Markdown source with syntax highlighting. Toggle freely between modes.

Full-Text Search

Instantly find any note with Tantivy-powered full-text search. Your entire library, always one keystroke away.

Plain Markdown Files

Notes are standard .md files in a folder you choose. No proprietary formats, no lock-in. Your files, your way.

Tables & Rich Content

Insert tables, task lists, code blocks with syntax highlighting, images, and more through slash commands or the format bar.

CJK-Latin Spacing

Automatic spacing between Chinese, Japanese, Korean, and Latin characters. Perfect typography for multilingual writing.

Keyboard-First

Command palette, slash commands, and comprehensive shortcuts. Navigate, format, and manage notes without leaving the keyboard.

Export Anywhere

Copy as Markdown, plain text, or HTML. Print to PDF. Export your writing in any format you need.

Live File Watching

Edit notes externally and Verso picks up changes instantly. Works seamlessly alongside Git, Obsidian, or any other tool.

Light & Dark Themes

Beautiful in any light. Automatically follows your system preference, or choose your own. Customizable typography settings.

Philosophy

Like Preview.app,
but for Markdown

Your disk is full of .md files — project docs, meeting notes, journals, READMEs. You shouldn't need a database or an app ecosystem just to read them.

Verso is a Markdown editor the way Preview is a PDF viewer: double-click a file, it opens, you read or edit, you close. No library to manage, no import, no sync.

Built with Rust and Tauri. Launches in under a second. Uses less memory than a browser tab.

Double-click .md → opens instantly in WYSIWYG
Or open a folder — browse and edit all files inside
Rust + Tauri — native speed, not Electron
No account, no cloud, no tracking, no internet needed
Open source, MIT licensed, free forever
